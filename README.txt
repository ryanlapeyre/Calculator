
Additional Features
* Resizable Interface that Scales 
* Equal Button Performs Last Equation 
* Decimal Input
* Negative Number Input
* Keyboard Keybinds
* Square Root Key
* Clear Button
* Maximize Keybind
* Quit Keybind
* Clean Interface 

Keybinds
* 0-9 are Mapped to NumPad, Traditional Numerical Keys
* Division : NumPad Divide Key, forwardslash (/)
* Multiplication : Star (*) on NumPad, Shift + 8
* Addition : NumPad Plus Key, Shift + Return (+)
* Subtraction : Numpad Minus Key, Minus on Keyboard (-)
* Equal Button : Return Key 
* Square Root Button : CTRL + Numpad Divide Key, Shift + forwardslash (/)
* Clear Button : Backspace, CTRL + Decimal on NumPad
* Maximize Button : F Key
* Decimal Button : Period on Keyboard, Decimal on NumPad
* Negative Button : Shift + Minus Key, CTRL + Minus on NumPad  (-)
* Quit Button : Q, CTRL + W

Bonus Included
* Negative Floating Point Numbers are Handled
